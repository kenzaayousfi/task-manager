# Task Manager

A web-based task management application developed in PHP and MySQL. The application allows users to create, manage and organize their daily tasks through a simple and intuitive interface.

## Features

- User registration and authentication
- Personal dashboard
- Create, edit and delete tasks
- User profile management
- Password update
- Secure session management

## Technologies

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository:

```bash
git clone https://github.com/kenzaayousfi/task-manager.git
```

2. Copy the project into your web server directory (`htdocs` for XAMPP or `www` for WAMP).

3. Create a MySQL database.

4. Configure the database connection in:

```
bdd/base_de_donnees.php
```

5. Start Apache and MySQL.

6. Open:

```
http://localhost/TO-DO-LIST/
```
